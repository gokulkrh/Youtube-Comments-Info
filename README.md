# Extracting Audience Feedback from comments using Opinion Mining and Text Summarization

This project extracts meaningful audience feedback from YouTube comments using NLP techniques like opinion mining and text summarization.

## Features

* Spam filtering using ML models
* Emotion detection using BERT-based models
* Keyword extraction (TF-IDF + TextRank)
* Extractive + abstractive summarization
* Web app using Flask API + React frontend

## Tech Stack

* Python (NLP, ML)
* BERT, T5
* Flask
* ReactJS
